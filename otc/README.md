# ELF OTC Bot

Telegram P2P гарант-бот для безопасных сделок.

## Возможности

- 💼 Создание сделок с выбором валюты
- 🛡️ Гарант-система для безопасности
- 📋 История сделок
- 👥 Реферальная система
- 🌐 Мультиязычность (RU/EN)
- 📊 Админ-панель

## Установка

1. Клонировать репозиторий
2. Установить зависимости: `pip install -r requirements.txt`
3. Скопировать `.env.example` в `.env` и заполнить
4. Запустить: `python app.py`

## Деплой на BotHost.ru

1. Создать проект на BotHost.ru
2. Подключить GitHub репозиторий
3. Добавить переменные окружения из `.env`
4. Запустить бота

## Технологии

- Python 3.12
- aiogram 3.x
- aiosqlite
- Telegram Bot API
